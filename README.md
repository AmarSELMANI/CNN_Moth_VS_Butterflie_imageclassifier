 Overview
This project was developed during my 3rd year (L3) Bachelor studies in 2025  
The goal is to build a Convolutional Neural Network to classify images into two classes:
- Butterfly
- Moth

The project focuses on applying deep learning techniques to a real-world image classification task.


 Dataset
Images were collected from a Kaggle dataset containing multiple butterfly and moth species.  
url : https://www.kaggle.com/datasets/gpiosenka/butterfly-images40-species?select=test
For binary classification:
- Species containing "moth" were labeled as Moth
- All others were labeled as Butterfly

---

 Method
- Image preprocessing and normalization  
- Data augmentation  
- CNN model built with **TensorFlow / Keras**
- Binary cross-entropy loss and Adam optimizer

---

 Results
The model successfully learned to distinguish between butterflies and moths, with stable training and validation performance.
